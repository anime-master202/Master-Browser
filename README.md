# Master-Browser
It's a Chromium-based Browser But it's ungoogled and 100$ private, doesn't leak your data like ip address just like brave does!!

Made With Love By Anime Master

How to run the browser with the good security patch?
Simply run the launcher.bat in the root of the browser folder
DON'T USE the the main exe in bin folder, it will just load but without our config which will make him like the other browsers
